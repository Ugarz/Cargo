# Let's do Rust
Getting Started Rust

# Process
Some checks

## Environnement
1. Check if you have installed rustc 
with `rustc --version`
2. Check if you have installed cargo with `cargo --version`

## Some commands

```bash
# Init a project
cargo new hello_world

# Compile Rust script
cargo build
# This will compile ur code in src/ to target/ folder
```
